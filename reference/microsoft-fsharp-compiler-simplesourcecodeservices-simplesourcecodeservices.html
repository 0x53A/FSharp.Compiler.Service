CompileToDynamicAssembly(...)
Signature:(otherFlags:string [] * execute:(TextWriter * TextWriter) option) -> ErrorInfo [] * int * Assembly option

Compiles to a dynamic assembly usinng the given flags. Any source files names are resolved via the FileSystem API. An output file name must be given by a -o flag, but this will not be written - instead a dynamic assembly will be created and loaded.

If the 'execute' parameter is given the entry points for the code are executed and the given TextWriters are used for the stdout and stderr streams respectively. In this case, a global setting is modified during the execution.
